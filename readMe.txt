Instructions on the data files used in this paper and the Matlab scripts (m-files) and functions used to analyse them. Data are held in CSV, MAT and XLS formats. CSV contain the same data as XLS file as comma-separated values and, note, NO data analysis takes place within those XLS files, they simply contain data. MAT files contain data that is, or has been, analysed using m-files.
